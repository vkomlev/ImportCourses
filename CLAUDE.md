Этот проект — часть портфолио, отслеживаемого в едином трекере `D:\Work\Root`
(ADR-0003, `tsk-050`). Полный протокол — `D:\Work\Root\CLAUDE.md`, секция

именно в этом проекте.

## Перед значимой работой

Новая фича, баг, рефакторинг — не рутинная правка:

1. Проверь `D:\Work\Root\tasks\_index.md` — нет ли уже задачи по теме.
2. Если нашлась — назови её номер (`tsk-NNN`) оператору перед началом.
3. Если нет и оператор дал явную формулировку («сделай X», «нужно Y», «почини Z») —
   заведи задачу:
   ```
   cd D:\Work\Root
   python -m orchestrator new-task --non-interactive --json \
     --title "..." --projects ParseCourse --directions "..." --tracks "..."
   ```
   (вывод — JSON, `PYTHONUTF8=1` перед командой на Windows-консоли, иначе кириллица
   в выводе будет искажена — сам файл всегда пишется в UTF-8 корректно).
4. Назови номер `tsk-NNN` оператору.

Это soft rule без технического принуждения — нарушение не ломает этот проект, но
рассинхронизирует единую точку управления портфолио
(`D:\Work\Root\docs\north-star.md`).
