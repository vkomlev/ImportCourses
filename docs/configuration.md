# Конфигурация

Все настройки парсера — константы в файле `src/config.py` (не переменные окружения). Меняются правкой файла перед запуском.

## Основные настройки

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| `MAIN_COURSE_UID` | `"PY"` | UID главного курса (можно переопределить флагом `--main-uid`) |
| `MAIN_COURSE_TITLE` | `None` | Название главного курса; если `None` — извлекается из `<title>`/`<h1>` страницы |
| `DEFAULT_ACCESS_LEVEL` | `"manual_check"` | Уровень доступа, записываемый во все курсы |

## Настройки HTTP-запросов

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| `USER_AGENT` | Chrome UA-строка | User-Agent для запросов к сайту |
| `REQUEST_TIMEOUT` | `30` | Таймаут запроса, сек |
| `RETRY_ATTEMPTS` | `3` | Число попыток при сетевой ошибке |
| `RETRY_DELAY` | `2` | Базовая пауза между попытками, сек (умножается на номер попытки) |

## Настройки разметки страниц

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| `H3_TAG` | `"h3"` | Тег заголовков подкурсов первого уровня на главной странице |
| `GO_LINK_TEXT` | `"Перейти"` | Текст ссылки перехода на страницу подкурса |
| `PLAN_SECTION_HEADER` | `"Краткий план раздела"` | Заголовок раздела со списком подкурсов уровня 2 |
| `MATERIALS_HEADER_TEXT_LESSONS` | `"Текстовые уроки"` | Заголовок секции текстовых материалов |
| `MATERIALS_HEADER_VIDEO_LESSONS` | `"Видеоуроки"` | Заголовок секции видеоматериалов |

## Настройки генерации UID

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| `UID_MAX_LENGTH` | `100` | Максимальная длина UID |
| `UID_SEPARATOR` | `"-"` | Разделитель частей UID |

## Настройки вывода

| Параметр | По умолчанию | Описание |
|----------|--------------|----------|
| `EXCEL_DEFAULT_FILENAME` | `"courses"` | Базовое имя файла курсов |
| `EXCEL_OUTPUT_DIR` | `"output"` | Каталог для XLSX-файлов |

Если сайт-источник использует другой текст заголовков/ссылок — правьте только этот файл, логика парсинга (`src/parser.py`) их не хардкодит.
